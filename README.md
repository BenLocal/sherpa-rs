# sherpa-rs

Rust bindings to [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx)

## Usage

```console
git clone --recursive https://github.com/thewh1teagle/sherpa-rs.git

cd sherpa-rs

cargo run --example speaker_id
```
